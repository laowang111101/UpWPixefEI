# 旅游管理系统 java1207

## 项目概述

旅游管理系统是一个基于SpringBoot和Vue.js的前后端分离架构的Web应用。

## 技术栈

### 前端

- **Vue.js**：用于构建动态交互界面。

### 后端

- **SpringBoot**：提供后端服务的运行框架。
- **Mybatis**：作为数据持久化层框架。

### 数据库

- **MySQL 5.x/8.x**：双版本适配，提供数据存储服务。

### 运行环境

- **JDK8**：Java开发工具包。
- **Maven**：项目管理和构建自动化工具。

## 功能模块

系统包括但不限于以下功能模块：

- 用户管理
- 订单管理
- 旅游产品管理
- 数据统计与分析

## 系统角色

- 系统管理员
- 普通用户
- 数据分析师

## 部署步骤

1. 准备JDK8和Maven环境。
2. 创建MySQL数据库，并导入数据库脚本。
3. 部署后端服务。
4. 部署前端应用。
